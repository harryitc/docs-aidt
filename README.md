# AIDT — Trung tâm Hỗ trợ

Trang tài liệu / trung tâm hỗ trợ của **Viện Trí tuệ Nhân tạo và Chuyển đổi số (AIDT) – HUTECH**, xây dựng bằng [VitePress](https://vitepress.dev).

Giao diện tiếng Việt, màu chủ đạo **xanh – cam**, hỗ trợ tìm kiếm và chế độ sáng/tối.

## 🚀 Bắt đầu

Yêu cầu **Node.js ≥ 18**.

```bash
npm install          # Cài đặt phụ thuộc
npm run docs:dev     # Chạy server phát triển → http://localhost:5173
npm run docs:build   # Build bản production ra docs/.vitepress/dist
npm run docs:preview # Xem thử bản production vừa build
```

## 📁 Thư mục chính

- `docs/` — toàn bộ nội dung tài liệu (Markdown).
- `docs/.vitepress/` — cấu hình site và giao diện (theme).
- `docs/public/` — tài nguyên tĩnh (ảnh, logo, favicon…).
